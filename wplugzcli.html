WPlugZ CLI

about

Made in C# by MF366, WPlugZ-CLI is the CLI tool for WriterClassic plugin developers.

It makes the plugin development process go from painful to easy, allowing you to focus on the code instead of the workspace.

It has tools such as Manifest File Verification, where the plugin's MANIFEST file is tested, in order to make sure it's a valid one.

I recommend you, dear user, to look at this project's GitHub repository in order to learn more about it.
